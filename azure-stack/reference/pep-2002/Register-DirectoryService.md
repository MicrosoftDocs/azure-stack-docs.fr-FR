# <a name="register-directoryservice"></a>Register-DirectoryService

## <a name="synopsis"></a>Synopsis
Script pour inscrire un annuaire client Active Directory (Azure AD) auprès du service Graph.
**CustomADGlobalCatalog** est le catalogue global pour un Active Directory personnalisé.
C’est la valeur `RootDomain` obtenue de la sortie générée par l’applet de commande **Get-ADForest**.

## <a name="syntax"></a>Syntaxe

```
Register-DirectoryService [-Force] [[-TimeoutInSecs] <Object>] [-SkipRootDomainValidation]
 [[-CustomADAdminCredential] <Object>] [-ValidateParameters] [[-CustomADGlobalCatalog] <Object>] [-AsJob]
```

## <a name="description"></a>Description
Script pour inscrire un annuaire client Active Directory auprès du service Graph.
**CustomADGlobalCatalog** est le catalogue global pour un Active Directory personnalisé.
C’est la valeur `RootDomain` obtenue de la sortie générée par l’applet de commande **Get-ADForest**.

## <a name="examples"></a>Exemples

### <a name="example-1"></a>Exemple 1
```
Register-DirectoryService -CustomADGlobalCatalog "contoso.com" -CustomADAdminCredential (Get-Credential) -TimeoutInSecs 1000
```
